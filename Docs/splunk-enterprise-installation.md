# Splunk Enterprise Installation

## Introduction

Splunk Enterprise is the SIEM platform used in this SOC Home Lab. It collects stores and displays logs from the Windows virtual machine.

I installed Splunk Enterprise on my host machine because it makes it easy to monitor Windows Event Logs in one place.

---

## Why I Chose Splunk

I selected Splunk Enterprise because it is one of the most widely used SIEM solutions in Security Operations Centers.

It helps security analysts search logs investigate security events and monitor systems.

---

## System Information

| Component | Details |
|----------|---------|
| Host Operating System | Windows 11 |
| Splunk Version | 10.4.1 |
| Installation Type | Local Installation |

---

## Installation Steps

### Step 1

Download Splunk Enterprise from the official Splunk website.

### Step 2

Run the installer as Administrator.

### Step 3

Accept the license agreement.

### Step 4

Create the administrator account.

### Step 5

Complete the installation.

### Step 6

Open Splunk using the following address.

```
http://localhost:8000
```

### Step 7

Sign in using the administrator account created during installation.

---

## Result

Splunk Enterprise was installed successfully.

The Splunk web interface opened without any issues and I was able to sign in using the administrator account.
