# Kali Linux Installation

## Introduction

Kali Linux is used as the second virtual machine in this SOC Home Lab.

I installed Kali Linux to build a complete lab environment with both Windows and Linux systems. It also helped me understand how to configure and manage multiple virtual machines.

---

## Why I Chose Kali Linux

I selected Kali Linux because it is one of the most popular Linux distributions used in cybersecurity.

It comes with many built in security tools and is widely used for learning and testing in lab environments.

---

## Virtual Machine Configuration

| Setting | Value |
|---------|-------|
| Operating System | Kali Linux |
| Memory | 4 GB |
| Processor | 2 CPUs |
| Storage | 50 GB |
| Network | NAT Network |

> **Note:** The virtual machine configuration may be different depending on your computer.

---

## Installation Steps

### Step 1

Download the Kali Linux ISO from the official Kali website.

### Step 2

Create a new virtual machine in Oracle VirtualBox.

### Step 3

Allocate memory processor and storage.

### Step 4

Attach the Kali Linux ISO file.

### Step 5

Start the virtual machine and complete the installation.

### Step 6

Create a local user account.

### Step 7

Restart the virtual machine after the installation is completed.

---

## Initial Configuration

After installing Kali Linux I verified that the operating system started successfully.

I also checked that the virtual machine had internet access and was connected to the virtual network.
